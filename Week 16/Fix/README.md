# Folder Fix - Perbaikan Website Submission Week 15

Folder ini berisi perbaikan dari setiap website yang telah dikumpulkan murid-murid.

## 📁 Struktur Folder

```
Fix/
├── README.md (file ini)
├── Jonathan-Kls-7/
│   ├── tentang-saya.html (diperbaiki dari HTML.html)
│   ├── style.css (diperbaiki)
│   └── PERBAIKAN.md (penjelasan perbaikan)
├── Justin-Arland/
│   ├── data-diri.html (diperbaiki dari DataDiri.html)
│   ├── style.css (diperbaiki)
│   └── PERBAIKAN.md (penjelasan perbaikan)
└── Nathan-Hans/
    ├── index.html (diperbaiki dari uhhhhhhhhh.html)
    ├── style.css (file baru)
    └── PERBAIKAN.md (penjelasan perbaikan)
```

## 🎯 Tujuan

Folder ini dibuat untuk:

1. ✅ Menunjukkan versi yang sudah diperbaiki
2. ✅ Membantu murid belajar dari kesalahan
3. ✅ Memberikan contoh best practices
4. ✅ Referensi untuk tugas selanjutnya

## 📚 Cara Menggunakan

1. Buka folder murid yang ingin dipelajari
2. Lihat file `PERBAIKAN.md` untuk tahu apa saja yang diperbaiki
3. Bandingkan dengan file asli di Week 15/submission
4. Pelajari perbedaannya
5. Terapkan di tugas kalian sendiri

---

**Catatan:** Ini adalah contoh perbaikan, bukan satu-satunya cara yang benar. Ada banyak cara lain yang juga baik!
